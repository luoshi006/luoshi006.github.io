# 3-Jekyll Theme

fork from zYeoman.


## 注意事项
使用时需要手动添加`index.html` `ie.md` `404.md` `tag.md`等文件，内容参考https://github.com/zYeoman/zYeoman.github.io

在`_config.yml`里添加

```yaml
remote_theme: zYeoman/3-Jekyll
```

详见 https://help.github.com/articles/adding-a-jekyll-theme-to-your-github-pages-site/

## Note

```
npm install --global sw-precache

sudo ln -s /usr/bin/nodejs /usr/bin/node
```
